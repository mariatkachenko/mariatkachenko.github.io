# Maria Tkachenko — Product Designer Portfolio

Dependency-free bilingual one-page portfolio built with HTML, CSS, and vanilla JavaScript.

## Local preview

From this directory, run:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Tests

```bash
node --test tests/site.test.mjs
```

If Node is not available in your shell, use the bundled Codex runtime path shown by `load_workspace_dependencies`.

## Deploy

The project can be uploaded as-is to any static host. The publish directory is the project root and no build command is required. Before using it as a real portfolio, replace the fictional name, email, social links, case studies, and metrics.
